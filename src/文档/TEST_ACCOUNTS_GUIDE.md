# 测试账号使用指南

## 📋 概述

本系统提供了完整的测试账号管理功能，让您可以快速创建和使用测试账号进行开发和测试。

## 🚀 快速开始

### 方法一：使用开发工具（推荐）

1. 在页面右下角找到紫色的 **齿轮图标** 浮动按钮
2. 点击按钮打开开发工具面板
3. 点击 **"🔧 测试账号管理"** 进入管理页面
4. 点击 **"初始化测试账号"** 按钮
5. 等待创建完成，查看结果

### 方法二：直接访问管理页面

1. 在浏览器中访问：`/test-accounts`
2. 点击 **"初始化测试账号"** 按钮
3. 等待创建完成

### 方法三：登录页面快速登录

1. 访问登录页面：`/login`
2. 找到 **"测试账号快速登录"** 黄色按钮
3. 点击展开测试账号列表
4. 首次使用时，点击 **"初始化测试账号"** 按钮
5. 初始化完成后，选择任意账号点击 **"快速登录"**

## ⚠️ 注意事项

1. **安全提醒**：
   - 这些测试账号仅用于开发和测试环境
   - 不要在生产环境使用测试账号
   - 测试账号的密码是公开的，请勿用于重要数据

2. **账号特性**：
   - 所有测试账号邮箱已自动验证
   - 每个账号都有预设的初始余额
   - 账号信息存储在 Supabase Auth 和 KV Store 中

3. **初始化说明**：
   - 如果账号已存在，初始化会跳过该账号
   - 初始化失败的账号会显示详细错误信息
   - 可以多次运行初始化命令

4. **开发工具**：
   - 开发工具浮动按钮仅在开发环境显示
   - 生产环境会自动隐藏

## 🔄 重置账号

如果需要重置某个账号：
1. 在 Supabase Dashboard 中手动删除用户
2. 重新运行初始化命令
3. 账号将使用初始配置重新创建
