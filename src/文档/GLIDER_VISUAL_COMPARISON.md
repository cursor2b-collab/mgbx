## 快速对比

### 旧版 vs 新版

| 特性 | 旧版（荧光绿） | 新版（液态玻璃） |
|------|----------------|------------------|
| **背景色** | `rgba(163, 240, 48, 0.25)` 荧光绿 | `rgba(255, 255, 255, 0.25)` 白色 |
| **模糊度** | `blur(10px)` | `blur(5px)` |
| **边框** | 无 | `1px rgba(255, 255, 255, 0.3)` |

## 品牌色使用

### 旧版
```
滑块背景：✅ 荧光绿
滑块边框：❌ 无
图标颜色：✅ 荧光绿（活跃）

```

品牌色占比：**高** (滑块本身就是品牌色)

### 新版
```
滑块背景：❌ 白色（中性）
滑块边框：❌ 白色
```

品牌色占比：**适中** (仅图标和文字)

**设计理念**：
- 滑块作为**背景元素**，使用中性色
- 图标和文字作为**前景元素**，使用品牌色
- 更清晰的视觉层次 ✨

## 技术实现差异

### 旧版（仅CSS）
```typescript
// 无悬停状态
const [gliderStyle, setGliderStyle] = useState(...);

// CSS完全控制
.glider-mobile {
    background: rgba(163, 240, 48, 0.25);
}
```

### 新版（CSS + React状态）
```typescript
// 追踪悬停
const [hoveredIndex, setHoveredIndex] = useState<number | null>(null);

// 事件处理
onMouseEnter={() => setHoveredIndex(0)}
onMouseLeave={() => setHoveredIndex(null)}

// 动态className
className={`glider-mobile ${hoveredIndex !== null ? 'glider-hovered' : ''}`}
```

## 浏览器兼容性

### backdrop-filter支持
| 浏览器 | 支持版本 | 备注 |
|--------|----------|------|
| Chrome | 76+ | ✅ 完全支持 |
| Safari | 9+ | ✅ 需要-webkit-前缀 |
| Firefox | 103+ | ✅ 完全支持 |
| Edge | 79+ | ✅ 完全支持 |

### 降级方案
```css
/* 现代浏览器 */
background: rgba(255, 255, 255, 0.25);
backdrop-filter: blur(5px);

/* 不支持backdrop-filter的浏览器 */
@supports not (backdrop-filter: blur(5px)) {
    background: rgba(255, 255, 255, 0.45); /* 增加不透明度补偿 */
}
```

## 切换建议

### 如果想要回到荧光绿滑块
恢复 `/styles/MobileBottomNav.css` 中的这些值：
```css
.glider-mobile {
    background: linear-gradient(135deg, 
        rgba(163, 240, 48, 0.25) 0%,
        rgba(163, 240, 48, 0.15) 100%);
    border: none;
    box-shadow: 
        0 0 20px rgba(163, 240, 48, 0.3),
        inset 0 0 20px rgba(163, 240, 48, 0.15),
        inset 0 1px 0 rgba(163, 240, 48, 0.4);
}

.glider-mobile.glider-animated {
    transition: 
        transform 0.5s cubic-bezier(0.34, 1.56, 0.64, 1) !important,
        width 0.5s cubic-bezier(0.34, 1.56, 0.64, 1) !important;
}
```

### 混合方案（荧光绿 + 悬停）
```css
.glider-mobile {
    background: rgba(163, 240, 48, 0.25);  /* 荧光绿 */
    backdrop-filter: blur(5px);            /* 毛玻璃 */
    border: 1px solid rgba(163, 240, 48, 0.3); /* 绿色边框 */
}

.glider-mobile.glider-hovered {
    background: rgba(163, 240, 48, 0.35);  /* 悬停变亮 */
}
```
